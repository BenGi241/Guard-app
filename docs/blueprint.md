# **App Name**: GuardDuty

## Core Features:

- User Authentication: Secure user login and authentication using a username/password system.
- Monthly Calendar View: Display a calendar for the current month showing available and reserved dates.
- Date Reservation: Allow users to reserve available dates for guard duty.
- Reservation Persistence: Store all date reservations and shift bookings
- Notification System: Send notifications to users upon successful reservation and eventual conflicts
- User Role: Assign role and limit access to date reseravtion feature

## Style Guidelines:

- Primary color: Use a muted blue (#6699CC) to convey trust and stability.
- Background color: Light gray (#F0F0F0) for a clean, professional look.
- Accent color: Green (#2E8B57) to indicate available slots.
- Font: 'PT Sans', a humanist sans-serif for both headlines and body text to ensure readability and a modern feel.
- Use simple, clear icons for calendar and reservation functions.
- Calendar should be the focal point, with clear indication of available and reserved dates.
- Subtle transitions when navigating between months or making reservations.
